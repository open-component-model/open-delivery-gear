---
name: Polish document
description: Review and update a text document to ensure it meets our quality standards.
---

This skill polishes a given text document.

# Invocation
```
/polish-docs path-to-doc
```

The `path-to-doc` argument can be either a fully qualified file path (relative or absolute), or a soft reference to a document within the same directory (including subdirectories) where the skill is invoked.

# Process

## Grammar and Typos

Review the entire text document.
Fix all grammar and spelling mistakes.

## British English

Ensure the document consistently uses British English.

## Polishing

Rephrase sentences to improve text quality.
Do not add or remove information.
Do not change semantics.
Focus purely on rephrasing for clarity and readability.

## Audience

The target audience has a software development background.
Use appropriate technical terminology for this audience.
